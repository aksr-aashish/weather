Weather App by Aashish Singh

How it Works

1.Enter City Name

\n\nType the name of the city you want to check the weather for in the search box.

2. Click the Search Button

After entering the city name, click the search button or press Enter to fetch the weather data.

3. View Weather Data

The weather data, including temperature, humidity, wind speed, and weather icon, will be displayed for the specified city.

Additional Information

The weather data is fetched from the OpenWeatherMap API using the user-entered city name. The API returns the current weather data, including temperature, humidity, wind speed, and weather icon.

The application uses modern CSS for styling and is responsive, ensuring a consistent user experience across different devices and screen sizes.
